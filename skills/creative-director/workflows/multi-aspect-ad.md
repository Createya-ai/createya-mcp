# Workflow: Multi-aspect ad — hero + кропы под платформы

**Триггер:** «один креатив для всех соцсетей», «hero + Stories + квадрат», «ad creative для разных платформ», «один концепт — несколько форматов»

**Continuity anchors:**
- Концепт целиком (субъект / композиция / свет / палитра)
- Текст / лозунг если есть

**Что меняется:**
- Aspect ratio
- Crop / framing
- Расположение текста (если есть) — под каждый формат

## Цепочка генераций

### Шаг 1 — Hero shot в основном aspect

**Главный кадр** обычно делается в **самом информационном** формате — обычно `4:5` или `16:9` (где помещается всё что нужно показать). Из него потом кропаем.

**Модель:** Nano Banana 2 / NB Pro / Flux 2 — зависит от стиля. Для постеров с текстом — Wan 2.7 Pro или Qwen Image 2 Pro.

**Промт строится с учётом будущих кропов:**
- Главный субъект — в центральной зоне (выживет в любом кропе)
- Negative space по краям — для текста / лого / CTA
- НЕ важные детали по углам (отрежутся в square)

### Шаг 2 — Approval и lock концепта

После approval — **запиши в session.md** все continuity anchors:
- Палитра (3 ключевых цвета)
- Light direction
- Стиль текста если есть
- Что НЕ меняется при кропах

### Шаг 3 — Генерация под каждый формат

Стандартный набор форматов:

| Формат | Aspect | Куда |
|--------|--------|------|
| **Square** | 1:1 (1024×1024) | Instagram feed, FB, основной |
| **Portrait** | 4:5 (1024×1280) | Instagram feed (новый стандарт), Facebook |
| **Story / Vertical** | 9:16 (1024×1820) | Stories, Reels, TikTok |
| **Landscape** | 16:9 (1820×1024) | YouTube thumb, OG / Twitter, web hero |
| **Wide** | 21:9 | Banner / billboard / cinematic |

**Для каждого формата** — свежая генерация **с тем же концептом**:
- `image_urls = [<approved_hero_cdn>]`
- Промт: `Same composition, same subject, same lighting, same palette as image 1. Reframe for [aspect] aspect ratio. Keep [главные элементы] visible and centered. Adjust [secondary elements] to fit new frame.`

**ИЛИ** — если модель поддерживает edit с новым `image_size` напрямую — генерим в нужном aspect сразу с anchor через `image_urls`.

### Шаг 4 — Vision QA каждого формата

- Главный субъект не обрезался криво
- Negative space сохранилась там где нужен текст
- Свет / палитра те же что в hero
- Композиция работает для нового aspect (не выглядит «обрезанной» — выглядит специально снятой)

## Что спрашивать

**Обязательно:**
1. Какие платформы? (выбор из списка форматов выше — multiSelect)
2. Есть ли текст / лозунг для размещения? (если да — где)
3. Главный субъект — фото есть или text-to-image?

**Опционально:**
- Бренд-цвета (HEX)
- Стиль / mood

**НЕ спрашивай:**
- Композиция — решаешь под каждый формат
- Освещение — единое во всей серии (lock после hero)

## Что НЕ меняется

```
LOCKED:
- Concept: [одна фраза]
- Subject: [что в центре]
- Lighting: [direction + quality]
- Palette: [3 HEX]
- Brand text: [если есть — текстовка дословно]
```

## Кредиты

- Hero: 1× стоимость модели
- 4 кропа: 4× стоимость
- Полный set из 5 форматов на NB 2 = ~50 cr, на NB Pro = ~90 cr

## Анти-паттерны

- ❌ Не делать просто «cropping» через imagemagick — у нас есть AI который понимает композицию
- ❌ Не перегенерировать концепт для каждого формата с нуля — потеряешь consistency
- ❌ Не пытаться засунуть весь контент в square — некоторые элементы должны жить ТОЛЬКО в landscape/wide
- ❌ Не забывать про negative space если текст приходит после
