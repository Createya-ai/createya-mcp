# Product artistic — художественная предметка с окружением

Mood-driven предметная съёмка. Товар вписан в атмосферную сцену, которая рассказывает историю. Для редакционных материалов, рекламных кампаний, lifestyle брендинга.

## Принцип
> "Товар — часть мира, а не центр вселенной. Сцена продаёт идею образа жизни, товар её воплощает."

Отличие от product hero: здесь есть narrative, окружение, атмосфера. Товар может быть не центрирован.

## Два основных mood-направления

### Direction 1 — Minimalist narrative

Чистые линии, мало элементов, максимальная breathing room. Silence и contemplation.

**Visual signature:**
- 2-3 предмета в кадре максимум
- Negative space работает как элемент композиции
- Light: natural window или single overhead directional
- Palette: monochromatic или analogous (2-3 тона)
- Surfaces: concrete, stone, linen, wood, paper

```
Mood markers: "minimalist still life, quiet luxury, contemplative atmosphere,
breathing room as design element, understated elegance"
```

**Composition rules:**
- Rule of thirds — product NOT центрирован
- Off-center placement с намеренным пустым пространством
- "Arranged with intentional negative space"

### Direction 2 — Moody atmospheric

Драма, тепло или холод, глубина. Несёт настроение — уют, ностальгия, дерзость.

**Visual signature:**
- Rich environment — текстуры, слои, детали
- Controlled chaos (carefully curated mess): "artfully arranged with complementary props"
- Light: warm candles / single practical / low-key
- Palette: rich warm (amber, cognac, terracotta) ИЛИ cool moody (deep blue, slate, forest)
- Surfaces: aged wood, velvet, marble, dark stone

```
Mood markers: "atmospheric still life, editorial moody, rich textured environment,
artfully curated scene, cinematic quality"
```

## Prompt structure (260+ слов)

```
[SCENE CONCEPT — 2 sentences]:
[What world this is: "morning ritual scene" / "evening aperitivo tableau" /
"artisan workspace vignette" / "romantic dinner preparation" etc.]
[Narrative: what just happened or about to happen]

[PRODUCT — 3 sentences]:
[Product name, description, material, color].
[Position in scene: "resting on aged oak surface" / "placed at left of composition" /
"partially obscured by surrounding elements"].
[How light falls on product specifically].

[SUPPORTING ELEMENTS / PROPS — 3-4 sentences]:
[Element 1: what, where, purpose].
[Element 2: complementary item that reinforces narrative].
[Element 3: optional texture layer — fabric, foliage, paper].
[Overall arrangement principle: "loosely arranged", "carefully composed", "lived-in natural"].

[ENVIRONMENT / SETTING — 3 sentences]:
[Surface: material, color, texture].
[Background: depth, what's visible, shallow DOF or mid-depth].
[Scale: tight vignette vs. environmental scene].

[LIGHTING — 3-4 sentences]:
[Source: window / single overhead / candles / practical lamp / natural outdoor].
[Quality: hard / soft / diffused / dappled].
[Color temperature: warm X Kelvin / cool / mixed].
[Shadows: direction, softness, role in composition].

Camera: [Sony A7R IV / Fuji GFX for film texture], [50mm f/2.8 for environmental /
35mm f/2 for immersive / 85mm f/2 for compressed still life].
Focus: [product sharp with creative out-of-focus elements / selective focus on key detail].

Color treatment: [muted and desaturated / rich warm tones / cool and airy / film-like].
Style: [minimalist editorial / moody atmospheric / romantic / raw artisan / cool graphic].
Quality: editorial still life photography, cinematic, high detail, photorealistic.

Negative: white studio background, even flat lighting, cheap stock photo feel,
excessive clutter, [specific to mood: "no cold lighting" for warm moody, etc.],
no text, no watermark.
```

## Palette presets

**Warm minimalist:**
```
"Warm cream tones, soft warm whites, muted terracotta accents, aged wood,
linen textures. Color temperature approximately 3500-4000K. Slightly desaturated."
```

**Cool editorial:**
```
"Cool grey tones, pale blue-grey surfaces, matte white, concrete texture,
colour temperature 5500-6000K. Graphic and clean."
```

**Rich moody:**
```
"Deep amber, cognac, burgundy accent tones, dark wood, aged surfaces.
Low ambient with warm practical light sources. Cinematic."
```

**Monochrome narrative:**
```
"Single-tone palette — all variations of one hue (beige / grey / green).
Tone-on-tone composition with texture variation. Sophisticated silence."
```

## Approval checkpoint

Для product artistic — больше творческой свободы, поэтому:
1. Обязательно показать юзеру concept brief (palette + scene concept) до генерации
2. Первый кадр — approval или direction correction
3. Только потом — серия вариаций
